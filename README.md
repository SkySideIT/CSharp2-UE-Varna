## ğŸ¯ Goal of the System

The system aims to automate the sales and delivery processes of the company **TechWorld AD**, which sells video games and offers courier services. The goals include:

- Automating product restocking

## ğŸ¢ Company Description

**TechWorld AD** is a video game retailer with multiple physical stores located in the Varna region. It serves individual customers and offers orders via phone, email, Skype, or a contact form. Sales can be split into multiple delivery routes, each carried out by company or customer couriers.

## ğŸ§© System Structure

- **Login screen**: User authentication with error handling  
- **Dashboard**: Main navigation panel after successful login
